# GServer
***
## Description
The back-end for big-graph demo.

## Install
`npm i`
## Start
`nodemon app.js`
## Usage
`localhost:3000/communityDetect    //detect community`
` localhost:3000/pageRank            //page rank`
` localhost:3000/shortestPath/1/2           //The shortest paths from node#1 to node#2.`
